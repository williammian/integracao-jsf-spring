#{jsfManagedBean.titulo} O problema disso é que não podemos usar os recursos do Spring dentro de um @ManagedBean.
Por exemplo, a propriedade "clienteService" está anotada com "@Autowired", mas como não é um bean Spring, então ele não injeta.
Veja o que é retornardo quando perguntamos "jsfManagedBean.clienteService eq null": #{jsfManagedBean.clienteService eq null}.
